Signature Orbit

Dictionary of all the CODX and CODY of the Signatures

URL Format:

With no signatures selected

/sign_orbit?start=start_data&stop=stop_date&data=cod_rebuilt

With 1 or + signatures selected

/sign_orbit?start=start_data&stop=stop_date&data=Signature1,Signature2...

Example:

With no signatures selected

http://cod-mon-api.lnls.br/sign_orbit?start=2023-05-11T07:30:00.000Z&stop=2023-05-11T09:30:00.000Z&data=cod_rebuilt

With 1 or + signatures selected

http://cod-mon-api.lnls.br/sign_orbit?start=2023-05-11T07:30:00.000Z&stop=2023-05-11T09:30:00.000Z&data=cod_rebuilt,SI-20M2:MA-SDB0_Y_S,SI-20M2:MA-SDB0_X_S

Returns:

Dictionary of the codx and cody of the selected signatures (cod_rebuilt always included)

Dictionary item structure:

Signature name: [ Array with the CODX, Array with the CODY ]

Signature Correlation

Dictionary of all the calculated correlations

URL Format:

/sign_comp?start=start_data&stop=stop_date

Example:

http://cod-mon-api.lnls.br/sign_comp?start=2023-04-11T07:30:00.000Z&stop=2023-04-11T09:30:00.000Z

Returns:

Dictionary of all the calculated correlations

Dictionary item structure:

Magnet Name: (Magnet type, Kick Axis, COD X, COD Y)

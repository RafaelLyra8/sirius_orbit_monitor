This folder consists of helpful static information for the website.

The main folders are:
    - constants
        -> date.ts -> Contain information about the intervals
        -> icon.ts -> Import a list of icons
        -> patterns -> Contain information about groups and modals
        -> pos.ts -> Name and order of the BPMs
        -> text.ts -> Consists of the static text for all the components
    - img -> Consist of the images used in the interface
    - interface -> Consist of all the specific interfaces and types built for the interface
    - style
        -> gen_styles.ts - Consist of styling patterns
        -> themes.ts - Consist of styling properties
    - types -> Consist of the type of the files used in the code

This folder consists of the controllers used to manipulate data on the website.

- archiver -> Interact by http with the Archiver Appliance
- bpm -> Functions related to the BPMs and the BPM Drift page
- chart -> Functions related to the chart
- orbit -> Functions related to the Orbit Drift page
- patterns -> General Functions
- simulation ->  Interact by http with the simulations on the backend
- time ->  Functions related to the dates

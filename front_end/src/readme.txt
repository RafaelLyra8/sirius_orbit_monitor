This is a website designed for the visualization of:
    - BPM (Beam Position Monitor) or Orbit drifts for the Sirius storage ring.
    - Total Orbit Correction applied to the storage ring.

The organization of this code is divided in several folders:
    - assets: Consists of helpful static information and patterns.
    - components: Consists of all the components used in the website.
    - data-access: Consists of all the interaction of the website with the EPICS Archiver Appliance.
    - controllers: Consists of helpful functions and information for the components.
    - pages: Consists of all of the pages of the website and the router function
    - redux: Consists of the information related to the react redux library
